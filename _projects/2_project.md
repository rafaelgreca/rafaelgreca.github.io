Repository dedicated to the developed solution for the end-of-course work of University of São Paulo's Master in Business Administration (MBA) in Artificial Intelligence and Big Data program. The solution main focus is to tackle the Speech Emotion Recognition task and is composed by a Convolutional Neural Network based on PANNs trained using Mel Spectrogram and Discrete Wavelet Transform.

The developed code can be found <a href="https://github.com/rafaelgreca/ser-wavelet">here</a>!
